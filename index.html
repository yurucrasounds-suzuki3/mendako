めんだこフィギュアをカメラで見つけて喋らせる

QRコードではなく、フィギュアそのものを見て喋らせる試作です。カメラ映像を画像分類モデルに通して、「めんだこ」と判定されたらブラウザ音声でセリフを喋ります。

待機中

このページがすること

  • ブラウザでカメラを起動
  • 読み込んだ画像分類モデルでフィギュアを判定
  • ブラウザ音声でセリフを喋る

設定

準備完了。カメラ開始を押してください。

使い方

  1. Teachable Machine などで「めんだこ / それ以外」の画像分類モデルを作る
  2. 公開されたモデルURLをこのページの「モデルURLのベース」に入れる
  3. index.html をChrome系ブラウザで開く
  4. モデル読み込み を押してから カメラ開始 を押す
  5. フィギュアを映して、めんだこ判定になったら喋る

この方式の特徴

  • フィギュアそのものを見て反応できる
  • 台座マーカーが不要
  • 角度や光の違いに対応するには学習画像が必要
  • スマホ単体より、まずPCやAndroid Chromeでの調整がやりやすい

おすすめ構成

  1. Teachable Machine で「めんだこ / それ以外」を学習する
  2. このページで判定しきい値を調整する
  3. セリフをランダム化する
  4. 必要なら録音済み音声に置き換える

補足

完全に「フィギュア単体だけ」で認識する場合は、光の当たり方、角度、背景で精度が落ちやすいです。正面だけでなく斜め、逆光、背景ありの写真も混ぜて学習させると安定しやすくなります。
