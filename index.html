めんだこフィギュアをカメラで見つけて喋らせる

まず動かすなら、フィギュア本体を直接画像認識させるより、台座か背面に小さなQRコードを貼って認識する方式が最も安定します。

待機中

このページがすること

  • ブラウザでカメラを起動
  • QRコードの文字列が mendako と一致したら
  • ブラウザ音声でセリフを喋る

設定

準備完了。カメラ開始を押してください。

使い方

  1. mendako という文字列のQRコードを作る
  2. それをフィギュアの台座か背面に貼る
  3. index.html をChrome系ブラウザで開く
  4. カメラ開始 を押す
  5. フィギュアを映すと喋る

なぜこの方式をすすめるか

  • 認識がかなり安定する
  • 学習データ作成が不要
  • スマホだけで試せる
  • 後から「見た目だけで認識」に置き換えやすい

次の段階

フィギュア自体の形や色で認識したいなら、次のどちらかです。

  1. ノーコード寄り: Google Teachable Machine で「めんだこ / それ以外」を学習し、学習済みモデルをWebページに組み込んで、検出時に speechSynthesis または録音音声を再生する
  2. しっかり作る: Python + OpenCV、カスタム物体検出モデル、音声合成APIや録音済み音声、必要なら口パク用ディスプレイやLED追加

おすすめ構成

  1. QR検出で喋る
  2. セリフをランダム化する
  3. 録音済みの声にする
  4. 画像認識に置き換える

補足

完全に「フィギュア単体だけ」で認識する場合は、光の当たり方、角度、背景で精度が落ちやすいです。展示やイベント用途なら、台座マーカー方式のほうが現実的です。
